 INTRODUCTION
 ------------
  - Custom module for creating products and showing QR code for purchasing the product.
  - Install pathauto and related modules for having customized URL pattern


 REQUIREMENTS
 ------------
  - Install codeitnowin/barcode package from packagist.org using: 
  - composer require codeitnowin/barcode


 INSTALLATION
 ------------

  - Install the Jugaad Purchase module as you would normally install a
  contributed Drupal module. Visit https://www.drupal.org/node/1897420 for
  further information.

  - Go to add content for product and update the data and
  - you can see the view of the product
